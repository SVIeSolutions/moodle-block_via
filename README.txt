The Via Bloc can only be used if you have the Via activity module.
This bloc creates shortcuts to the Via recordings that have been shared within the course.

To Install
1. Copy the progress directory to the blocks directory of your Moodle instance
2. Visit the notifications page

For more information, visit...
http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

Once the bloc is installed, you can use it in a course as follows.

1. Turn editing on
2. Add the Via block to your page (http://docs.moodle.org/en/Blocks) where you wish it to appear.
3. Create your Via activities as normal
4. Any recorded Live session that is shared will appear atomatically in the list.
